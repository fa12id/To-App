    Panduan Aplikasi

    • ➕ Tambah tugas: isi teks, pilih tanggal/jam, set notifikasi (opsional), lalu klik tambah.
    • ✔ Tandai selesai: klik tombol centang di samping tugas.
    • ✏️ Edit: klik tombol pensil untuk ubah nama tugas.
    • ✖ Hapus: klik tombol silang untuk hapus tugas.
    • 📊 Progress: lingkaran menunjukkan jumlah selesai / total tugas.
    • 🔍 Cari: gunakan kotak search untuk filter tugas dengan cepat.
    • ⚪ Filter: pilih untuk menampilkan semua / aktif / selesai saja.
    • 🔔 Notifikasi: akan muncul sebelum deadline sesuai waktu yang diatur.
